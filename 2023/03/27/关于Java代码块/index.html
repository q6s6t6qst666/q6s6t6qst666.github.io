static代码块和构造代码块

结论:static代码块伴随着类的加载，只在类第一次被加载的时候调用一次
构造代码块是用来给新建的对象进行初始化的，构造几个对象，就调用多少次，是通用的
局部代码块是用于特定局部的一种代码，在执行完局部代码之后，就被释放掉了

构造代码块和构造方法

结论:
构造代码块
特点：对象一建立就运行了，而且优先于构造函数执行
作用：给对象进行初始化
构造代码块与构造方法区别：
构造代码块是给所有的对象进行统一的初始化(每次创建对象必调用)
构造方法是对应的对象进行初始化

(静态)内部类/方法

静态内部类/方法 只能访问外部类中的静态成员，要访问非静态成员，必须依赖于外部类的实例对象
eg:

public class test11_student {
    class Test{
        int num;
        public int getNum(){
            return num;
        }
    }
    public static void main(String[] args) {
        test11_student test = new test11_student();
        Test a = test.new Test();
        a.num = 10;
        System.out.println(a.getNum());
    }
}
