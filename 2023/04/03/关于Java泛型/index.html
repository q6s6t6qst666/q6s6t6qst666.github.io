关于Java泛型

发表于2023-04-03|更新于2023-04-11|编程java
|阅读量:

泛型概念

  • 使用类型参数，使程序更为通用，适用于处理不同类型的数据
  • 泛型的本质是参数化类型，即所操作的数据类型被指定为一个参数
  • 可以声明泛型类、泛型方法和泛型接口
  • 没有泛型的时候使用集合 输出结果
  • 有泛型的情况下使用集合 有了泛型后，定义好的集合names在编译的时候add(123)就会编译不通过
    相当于告诉编译器每个集合接收的对象类型是什么，编译器在编译期就会做类型检查，告知是否插入了错误类型的对象，使得程序更加安全，增强了程序的健壮性

泛型中KTVE的含义

  • T: Type任意类型
  • E：Element (在集合中使用，因为集合中存放的是元素)
  • K：Key（键）
  • V：Value（值）
  • N：Number（数值类型）N：Number（数值类型）
  • ？：表示不确定的java类型

注意： 泛型类不支持基本数据类型
